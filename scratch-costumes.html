How-to: Edit costumes in Scratch 3

Open Scratch, and either create a new project, or go to the project you wish to edit.

Navigate to the Costumes tab, found in the top left (you will only see this if you are writing code for the sprite, not the background).


Right click on the costume you want to edit, and choose duplicate.

Now any changes you make will not effect the original.


You can use the buttons above and on the left of the image to edit the costume. For example, changing the colour, size or deleting parts of the image.

Rename your costume, so you can easily find it later.


To use your costume in your program, navigate to the Code tab, found in the top left, and use the Looks blocks in your program.
